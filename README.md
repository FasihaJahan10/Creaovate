# Creovate AI Website

This project consists of a FastAPI backend and a React (Vite) frontend.

## Prerequisites
- Python 3.8+
- Node.js & npm

## Setup & Running

### 1. Backend (Python)
Navigate to the root directory:
```bash
cd c:\Users\fasih\PycharmProjects\Creaovate-backend
```

Install dependencies:
```bash
pip install -r requirements.txt
```

Start the backend server:
```bash
uvicorn main:app --reload
```
The backend will run at `http://localhost:8000`.

### 2. Frontend (React)
Open a new terminal and navigate to the frontend directory:
```bash
cd c:\Users\fasih\PycharmProjects\Creaovate-backend\frontend
```

Start the development server:
```bash
npm run dev
```

### 3. Access the Website
Open your browser and visit:
**http://localhost:5173**

## Features
- **Brand Generator**: Create unique brand names.
- **Content Creator**: Generate marketing copy.
- **Sentiment Analysis**: Analyze text emotion.
- **Chatbot**: Chat with Creovate AI.
- **Logo Concepts**: Generate logo design prompts.
